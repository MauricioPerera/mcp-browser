MCP Browser

A browser with a built-in MCP server. Navigate to any page — agents get auto-tools.

MCP Endpoint

POST http://127.0.0.1:3002/mcp

Auto-registered tools (every page gets these)

page_title     — Get the page title
page_url       — Get the current URL
page_text      — Get page text content (first 5000 chars)
page_html      — Get page HTML (first 10000 chars)
query_selector — Query DOM elements by CSS selector
navigate       — Navigate to a URL

Custom tools (pages can register their own)

<script>
  mcpTool("search", "Search products", { query: { type: "string" } },
    (args) => searchProducts(args.query)
  );
</script>

Test from terminal

curl -X POST http://127.0.0.1:3002/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list","params":{}}'
